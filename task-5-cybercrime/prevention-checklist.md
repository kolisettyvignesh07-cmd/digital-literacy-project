# Cybercrime Prevention Checklist

## Safety Measures

-> Use strong and unique passwords for all accounts
-> Enable two-factor authentication (2FA) wherever possible
-> Never share OTPs, passwords, or PINs with anyone
-> Avoid clicking on suspicious links or messages from unknown sources
-> Always verify website URLs before entering personal information
-> Keep devices, apps, and software updated regularly
-> Install antivirus and basic security tools on your devices
-> Do not download files or apps from untrusted sources
-> Be cautious while using public Wi-Fi networks
-> Regularly monitor bank and online accounts for unusual activity
-> Never approve unknown UPI collect or payment requests
-> Double-check the receiver’s name before making any UPI transaction
-> Avoid scanning random or unknown QR codes
-> Never share bank details or UPI PIN with anyone
-> Report cybercrime on the National Cyber Crime Portal (**cybercrime.gov.in**)
-> In case of financial fraud, immediately **call the helpline 1930**

## Conclusion
Following these simple precautions can help protect against cybercrime and ensure safe internet usage.
